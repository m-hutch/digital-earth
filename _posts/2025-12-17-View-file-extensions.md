# Viewing file extensions
A **file extension** is text at the end of a file name that lets the computer know what type of file it is. During this class, it will be helpful if you can see the file extension too. Use these videos to help you turn this setting on for your operating system.

- [Windows 11: Show File Extensions](https://youtube.com/shorts/uvjO80zMUUs){:target="_blank"}
